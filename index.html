  1. HTML Notes

    Element Tags

    • HTML stands for Hypertext Markup Language. Hypertext refers to a text that links to other text documents. Markup Language refers to the Tags that denote elements of the html text.
       
    • HTML files are text files save as name.html. Generally the first file in a folder of html files for a project is named index.html. The first line of code in an HTML file is <!DOCTYPE HTML> this denotes the type of HTML we are using HTML5. The second tag in html is the <html> tag and the last tag in a html document is the <html> tag.
       
    • Example of basic HTML File:

      <!DOCTYPE HTML>
      <html>
          <head>
              <title>Code Example</title>
          </head>
          <p>This is the text.</p>
      </html>
    • There are a few common tags used for a basic html document.
       
      • <head> HTML element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets. Note: <head> primarily holds information for machine processing, not human-readability.
         
      • <title> tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

         
      • <div> tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements. You can add an attribute ID and give it a name to call the that division.
         
      • <p> HTML element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.
         
      • <a>: The Anchor element. The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Examples: href=”google.com”, local repository href=”./file.type”, html ID href=”#IDname”.
         
      • <!--...--> The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.
         
      • <table> The Table tag allows the creation of tables

    Debugging

    • The Inspector: DOM explorer and CSS editor. The developer tools usually open by default to the inspector, which looks something like the following screenshot. This tool shows what the HTML on your page looks like at runtime, as well as what CSS is applied to each element on the page. It also allows you to instantly modify the HTML and CSS and see the results of your changes reflected live in the browser viewport. Press F12 or right click and select inspect.
