HTML Notes and Links.

  1. HTML Notes

    • HTML stands for Hypertext Markup Language. Hypertext refers to a text that links to other text documents. Markup Language is in refers to the Tags that denote elements of the html text.
    • HTML files are text files save as (name.html). Generally the first file in a folder of html files for a project is named (index.html). The first line of code in an HTML file is <!DOCTYPE HTML> this denotes the type of HTML we are using HTML5.
    • Example of basic HTML File:

      <!DOCTYPE HTML>
      <html>
          <head>
              <title>Code Example</title>
          </head>
          <p>This is the text.</p>
      </html>
